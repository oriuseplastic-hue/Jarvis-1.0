# Jarvis-1.0 — Personal assistant-style project

Jarvis-1.0 is a small assistant-style project that showcases features and demos. This repository contains the source, demo, and documentation so others can try Jarvis-1.0 locally or on the demo site.

Features
- It can Search your command in Youtube or Wikipedia.
- It can Auto type anything.
- It can translate any languages to English[EN-Us].
- It can press F11[Key].
- It can email your Friend.
- It can Auto Correct your said spelling.
- It can open Softwares.
- It can Shutdown,SleepMode and Restart your Computer.
- It can tell the time.
- It can tell the weather.
- It can open all the sites.
- It can show you the sites list.

Quickstart (generic)
1. Clone the repo:
   ```git clone https://github.com/oriuseplastic-hue/Jarvis-1.0.git```
2. Change directory:
   ```cd Jarvis-1.0```
3. If there is a package.json:
   ```npm install```
   ```npm start```
4. Install All the library's:
   ```pip install -r requirements.txt```
5. Run the Code:
   ```python main.py```
